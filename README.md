# hubot-oden

A hubot script that does the things

See [`src/oden.coffee`](src/oden.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-oden --save`

Then add **hubot-oden** to your `external-scripts.json`:

```json
[
  "hubot-oden"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
